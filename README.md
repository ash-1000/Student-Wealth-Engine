# 💰 Student Wealth Lab

Build a smart portfolio, understand risk, and explore real market data — designed specifically for students.

---

## 🚀 Features

* 📊 **Portfolio Builder**

  * Select stocks and assign weights (must total 100%)
  * Visualize allocation with a pie chart

* 📈 **Wealth Growth Simulator**

  * Calculates future wealth based on monthly investments
  * Uses real stock data (CAGR-based returns)

* 🔥 **Risk Indicator**

  * Conservative 🛡️
  * Balanced ⚖️
  * Aggressive 🚀

* 🧠 **Portfolio Insights**

  * Detects index funds vs individual stocks
  * Provides simple, educational explanations

* 📉 **Stock Explorer**

  * View 1-year performance of selected stocks

---

## 🛠️ Tech Stack

* Python
* Streamlit
* yfinance
* matplotlib

---

## 📦 Installation

Install required libraries:

```bash
pip install streamlit yfinance matplotlib
```

---

## ▶️ Run the App

Make sure you're in the project folder, then run:

```bash
python -m streamlit run finance_app.py
```
